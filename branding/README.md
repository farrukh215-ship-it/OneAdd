# TGMG Brand Kit

Available master assets:

- `TGMG-mark.svg` - icon only (light background)
- `TGMG-logo-light.svg` - full logo for light backgrounds
- `TGMG-logo-dark.svg` - full logo for dark backgrounds
- `TGMG-logo-mono.svg` - monochrome full logo
- `TGMG-favicon.svg` - favicon-safe icon

Usage targets:

- `apps/web/public/brand/*`
- `apps/admin/public/brand/*`
- `apps/mobile/assets/TGMG-logo.jpg` (Expo icon/splash fallback)

Notes:

- SVG is source-of-truth for web/admin.
- For Play Store marketing assets, export PNG variants from these SVG files at 512, 1024, and 2048 px.
